## Patient Dashboard App
Simple Android healthcare dashboard app that displays patient information and medical details.

## What it does
Shows patient profile and vital signs
Opens detailed medical record screen
Provides quick medical action buttons
Launches emergency dialer
Includes bottom navigation menu

## Files
MainActivity.java – dashboard logic
PatientDetailsActivity.java – detailed patient screen
activity_main.xml – main dashboard layout
activity_patient_details.xml – patient record layout
AndroidManifest.xml – app configuration

Built with Java and XML using the Android SDK.
